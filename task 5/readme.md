# Task 5 - React Frontend Project

## 📌 Project Overview

This project is developed using **React.js** with **Vite** as part of the MERN Stack Internship tasks.

The project demonstrates the implementation of a component-based React application with a responsive user interface including a navigation bar, registration form, and footer section.

- JavaScript (ES6)

## ⚙️ Installation and Setup

Follow these steps to run the project locally.

### Step 1: Open Project Folder

Open the project folder in VS Code.

### Step 2: Install Dependencies

Run the following command in the terminal:

```bash
npm install
```

This installs all required dependencies.

---

## ▶️ Running the Application

Start the development server using:

```bash
npm run dev
```

The application will start on:

```
http://localhost:5173/
```

Open the link in your browser to view the project.
