# VIT MERN Internship Tasks

This repository contains the tasks completed as part of the VIT Advanced MERN full stack course.
## Repository Structure

### Task-1: Feedback Form
A customer feedback survey form created using HTML.

**Features:**
- Name input fields
- Email input
- Star rating section
- Satisfaction survey using radio buttons
- Recommendation survey

---

### Task-2: User Information Form
A user registration form designed using HTML and CSS.

**Features:**
- Full Name
- Email
- Password & Confirm Password
- Gender Selection
- Date of Birth
- Address
- Submit Button

---

### Task-3: Mini Educational Institute Website

A multi-page educational institute website built using HTML and CSS.

### Pages Included

#### Home Page
- Navigation Bar
- Institute Banner
- Welcome Message
- Video Integration
- Audio Integration

#### Courses Page
- Web Development Courses
- Programming Language Courses
- Ordered and Unordered Lists

#### Schedule Page
- Course Schedule Table
- Trainer Information
- Course Duration
- Mode of Learning

#### Registration Page
- Student Registration Form
- Course Selection
- Gender Selection
- Message Box
- Register Button

---

## Technologies Used

- HTML5
